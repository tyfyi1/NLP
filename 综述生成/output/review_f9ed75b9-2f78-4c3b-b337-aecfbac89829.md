# 大规模预训练模型在生物信息学中的前沿进展：从基础模型构建到高效下游适应

## 摘要

随着高通量测序技术的飞速发展，生物信息学领域正经历着一场由大规模预训练模型（Large-scale Pre-trained Models）驱动的范式变革。这些模型，通常被称为基础模型（Foundation Models）或大语言模型（Large Language Models, LLMs），通过在海量生物序列和单细胞转录组数据上进行自监督学习，成功捕捉了生命系统的底层规律。本文系统综述了该领域的最新研究进展，重点聚焦于两大核心主题：一是面向单细胞转录组学和网络生物学的大规模基础模型构建；二是参数高效微调（Parameter-Efficient Fine-Tuning, PEFT）等创新策略在下游任务适应中的应用。文章梳理了从模型架构创新到具体生物学问题求解的完整研究脉络，分析了当前模型在数据整合、计算效率及跨模态融合等方面面临的挑战，并展望了未来向多模态、可解释、高效能生物智能系统发展的趋势。

## 1. 引言

生物信息学的核心任务之一是从海量、高维、高噪声的生物数据中解码生命活动的分子机制。传统计算方法往往受限于特定任务的数据规模和特征工程的复杂性。近年来，受自然语言处理领域大语言模型成功的启发，研究者开始探索将大规模预训练模型应用于生物序列与单细胞组学数据，以期学习更具泛化能力的生物表示 [2]。这一趋势的核心理念在于，通过在海量无标签数据上进行预训练，模型能够习得基因调控网络、蛋白质功能以及细胞状态的内在规律，从而以“迁移学习”的方式高效赋能各类下游任务，尤其为数据稀缺场景（如罕见病研究）提供了突破性解决方案 [5]。

然而，将大规模预训练模型成功应用于生物信息学仍面临诸多挑战。首先，生物数据具有独特的模态特性，例如单细胞转录组数据的高维度（约两万个基因）和测序深度差异 [6]。其次，通用预训练模型在零样本（zero-shot）设置下针对特定生物任务的性能往往不佳，而传统的全量微调又存在灾难性遗忘和高昂计算成本的问题 [1]。因此，如何构建契合生物数据特性的基础模型，并开发高效的下游任务适应方法，成为当前研究的焦点。

本文旨在系统综述大规模预训练模型在生物信息学中的最新进展。文章主体将围绕两条主线展开：第一部分介绍面向单细胞转录组学与网络生物学的基础模型构建，重点分析Geneformer和scFoundation等代表性工作；第二部分探讨以参数高效微调为代表的下游任务适应技术，并展示其在细胞类型鉴定、蛋白质功能预测和基因调控关系识别等任务中的应用。最后，本文将总结当前研究的共性趋势与局限，并展望未来发展方向。

## 2. 大规模生物基础模型的构建范式

构建能够理解生命系统复杂性的基础模型，需要从数据、模型架构和预训练任务三个维度进行系统性创新。当前研究主要沿两条路径展开：一是以基因网络动态建模为目标的上下文感知模型，二是以海量单细胞转录组为对象的超大规模生成式模型。

### 2.1 面向网络生物学的上下文感知预训练

基因调控网络并非静态不变，而是高度依赖于细胞上下文环境。为了捕捉这一动态特性，Theodoris等人开发了Geneformer，这是一个在约2990万个人类单细胞转录组语料库（Genecorpus-30M）上预训练的上下文感知Transformer模型 [5]。其核心创新在于采用自监督掩码学习目标，使模型在预训练过程中自发地学习到基因网络的层次结构。研究发现，Geneformer的注意力权重能够编码转录因子和核心调控节点的相对重要性，这为理解基因剂量敏感性和染色质动态提供了全新视角。更重要的是，该工作开创性地将迁移学习范式引入网络生物学，使得在仅有极少量任务特异性数据（如884个细胞）的情况下，仍能实现精准预测，从而极大地推动了罕见病机制研究和候选治疗靶点的发现 [5]。

### 2.2 面向海量单细胞转录组的超大规模生成式建模

单细胞RNA测序数据的高维度（近两万个基因）和测序深度差异，对模型架构的可扩展性提出了严峻挑战。Hao等人提出的scFoundation模型，通过设计名为xTrimoGene的非对称编码器-解码器架构，成功构建了一个拥有1亿参数、在超过5000万个细胞上预训练的超大规模单细胞转录组基础模型 [6]。该工作的关键突破在于提出了“读长感知”（Read-Depth-Aware, RDA）预训练任务。通过引入目标总计数和源总计数指标，模型不仅能学习基因间的共表达关系，还能有效协调不同测序深度带来的技术差异，从而赋予了模型无需微调即可增强低质量数据基因表达的能力。实验证明，scFoundation在基因表达增强、药物反应预测和单细胞扰动预测等多个下游任务中均展现出卓越性能，其验证损失随模型规模增加呈现幂律下降，符合神经网络的“扩展法则”，充分证明了超大规模预训练在单细胞生物学领域的巨大潜力 [6]。

## 3. 高效下游任务适应与多领域应用

预训练基础模型提供了通用的生物表示，但如何将其高效、精准地应用于特定下游任务，是决定其应用价值的关键。当前研究一方面致力于开发更优的模型适应策略，另一方面也在不断拓展其应用边界至蛋白质功能预测和基因调控关系识别等领域。

### 3.1 参数高效微调：平衡性能与计算成本

针对单细胞大语言模型在零样本设置下表现不佳，而全量微调又面临灾难性遗忘和高计算成本的双重困境，He等人首次将参数高效微调（PEFT）策略引入单细胞领域，以scGPT模型为基础进行了系统验证 [1]。研究提出了两种新颖的PEFT策略：“基因令牌提示”（Gene token prompt）在嵌入层之上设计适配器层以对齐数据分布；“基因编码器提示”（Gene encoder prompt）则在Transformer编码器内部引入适配器，将学习到的基因关系投影到自适应子空间。在四个公开单细胞数据集上的实验结果表明，PEFT策略不仅在准确率、F1分数上一致优于全量微调，而且可训练参数量减少了高达90%。特别地，该方法在识别稀有细胞类型和难以区分的细胞亚型（如初始T细胞与记忆T细胞）方面展现出显著增强的能力，为在计算资源受限条件下高效利用大规模预训练模型提供了可行方案 [1]。

### 3.2 蛋白质功能预测中的多源语言模型整合

在蛋白质-DNA结合位点预测这一关键任务中，Zhu等人提出的ULDNA模型展示了整合多源预训练语言模型的巨大潜力 [3]。该模型创新性地融合了在UniRef50、BFD和UniClust30三个不同大规模序列数据库上预训练的ESM2、ProtTrans和ESM-MSA蛋白质语言模型，以提取具有进化多样性的互补特征嵌入。随后，通过一个专门设计的LSTM-Attention网络解码这些特征，强化残基级别的进化特征与DNA结合模式之间的关联。在多个基准数据集上的系统测试表明，ULDNA的预测精度显著优于17种现有方法，其马修斯相关系数（MCC）在独立测试集上比次优方法平均提高了13.3%。该研究不仅证明了多源语言模型整合策略的有效性，也为仅从蛋白质序列出发进行高精度功能预测开辟了新路径 [3]。

### 3.3 基因调控关系识别的基准测试

除了对专用生物模型的探索，评估通用型预训练大语言模型在生物任务上的潜力同样重要。Noel等人首次系统性地基准测试了Claude 3.5 Sonnet、GPT-4o、Gemini 1.0 Pro等通用型LLM在识别转录因子（TF）与靶基因调控关系及其调控类型（激活、抑制）上的表现 [4]。研究使用了文献驱动（TRRUST v2）和实验驱动（TFLink）两类数据集，并深入分析了提示工程、温度参数等因素的影响。结果表明，无需领域特异性微调，通过精心的提示设计，通用LLM即可在TF-靶标识别任务上取得合理表现，其中Claude 3.5 Sonnet在实验数据上的平衡准确率达到72.6%。该研究为缺乏专业计算技能的研究人员提供了一种低门槛、易获取的调控生物学研究辅助工具，并强调了提示工程在释放LLM效能中的关键作用 [4]。

## 4. 结论与展望

大规模预训练模型正在重塑生物信息学的研究范式。从Geneformer和scFoundation等专用基础模型的构建，到PEFT等高效适应策略的开发，再到多源语言模型整合与通用LLM基准测试，当前研究已清晰地勾勒出一条“大规模预训练—高效下游适应—多领域应用赋能”的技术路径 [?]。这些工作共同证明，从海量生物数据中学习通用表示不仅可行，而且能够显著提升细胞类型鉴定、药物靶点发现、蛋白质功能预测等关键任务的性能，尤其为数据稀缺的罕见病研究带来了变革性工具。

然而，该领域仍处于发展初期，面临若干共性挑战。首先，数据质量与整合仍是瓶颈，如何有效融合跨实验平台、跨组织、跨物种的多源异构数据，是构建更强大基础模型的前提 [2]。其次，计算效率问题突出，尽管PEFT策略已大幅降低适应成本，但预训练阶段的算力消耗依然巨大，探索更高效的模型架构和训练策略势在必行 [6]。再者，当前模型大多局限于单模态（如仅转录组或仅蛋白质序列），缺乏对多组学、影像、文本等多模态信息的联合建模能力。

展望未来，生物信息学基础模型的发展将呈现以下趋势：第一，多模态与混合架构的融合，通过整合知识图谱、物理约束与多组学数据，构建更全面、更可靠的生命系统解码器 [2]。第二，可解释性的增强，发展新的分析方法以理解模型内部学到的生物学规律，使其从“黑箱”走向“白箱”，从而加速科学发现 [5]。第三，临床与转化应用的深化，将模型应用于精准医学、药物重定位和个性化治疗等实际场景，真正实现从计算模型到临床价值的转化 [4]。随着算法、算力和数据的持续演进，大规模预训练模型有望成为解码生命奥秘、推动精准医学发展的核心引擎。
